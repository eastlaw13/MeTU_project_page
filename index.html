Project Overview

MeTU addresses the trade-off between accuracy and efficiency by integrating MobileViT as a backbone with U-Net style skip connections. This hybrid approach leverages global context from Transformers while preserving spatial details via CNN skip connections, optimized for real-time inference on edge devices like Raspberry Pi.

Quantitative Evaluation: Cityscapes

MeTU-xs achieves 73.46% mIoU on Cityscapes, outperforming Segformer-b0 despite using 45% fewer parameters.

Evaluated on Cityscapes-C, MeTU-xs maintains a stable 43.87% mPC, proving its reliability under environmental corruptions compared to traditional ASPP-based decoders.
